Design 1

Home:

  • logged in user has probably used the application before and does not need to see the “about” section, this is probably more applicable to first time users on the main page before logging in
  • the news is interesting to see how other people are using the application and what is being used the most – might be interesting to see relevant “news” to communities user is following
  • I would imagine the “home” page showing information about saved searches or information about joined communities – this is the information I would think the user would most likely be navigating to


View

  • This section might more appropriately named, “view” is a little ambiguous. Perhaps “Communities”?
  • It would be great to see a view of a specific Community or “Search” result – that is probably one of the more important components to this site


Create

  • Looks good, may need more filters under website – specify what information is automatically be grabbed and what the user can specify?


Edit

  • I think this shouldn’t be shown as a tab but as a function while viewing a specific Community or more likely a Website
  • I don’t understand why you would be moving websites between communities, or perhaps they’re shared/linked? Would the information gathered from that website for that specific community be pertinent to the other community?


Users

  • Depending on the audience who would use this tool, this section might be most relevant to be shown per community. If it’s more for a business use case, I would imagine it would have less of a generic social networking vibe.

 

Design 2

Creator

  • Perhaps rename to “Create” or “New”, similar for the other tabs (e.g. “Edit” rather than “Editor”)
  • Recently created communities may go under the news section or the homepage.


My Communities

  • it’s unclear as to whether Websites is a subcategory of a Community given the way this is laid out
  • it may be redundant to show which Community and Website is selected and also write it
  • I like the graph being displayed there, it would be interesting to add information to edit it on the fly – adding additional statistics, keywords, etc.


Browser

  • I think this would make more sense if this was a spot that you could drill down into the Communities – given the name “Browser” - perhaps should be combined with the “My Communities” tab
  • Internet Communities is a confusing name – maybe “All Communities”?
