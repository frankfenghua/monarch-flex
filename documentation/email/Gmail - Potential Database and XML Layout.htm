In some test scripts I downloaded all the links on the following site:

And then I started a new thread for each url (threaded.py):

user    0m0.452s

Results can vary quite a bit but the serial approach was generally about 5 times slower than the concurrent one.

It seemed like I was able to download a group of pages very quickly, and then I would have to wait awhile until the next bunch would download.
